# FlowGate v1.0.3

本版本重点修复 Step 6 的 PayPal 验证码恢复逻辑，避免验证码错误后直接回到流程开头。

## 更新内容

- PayPal 验证码错误时，优先留在当前验证码页点击 Resend，并重新拉取验证码。
- 连续失败 3 次后，才切换号码继续当前页面，而不是直接重启整条流程。
- 调整验证码获取流程，避免每次拉码就清空当前号码源。
- 同步修正相关错误提示文案，便于排查验证码接口返回异常。

## 说明

- 如果当前验证码页提示 `Get a new code`，系统会在当前页内重试。
- 如果验证码接口暂时取不到码，会继续在当前页面重试，而不是回到第一步。
