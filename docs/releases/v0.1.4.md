# FlowGate v0.1.4

历史版本记录，已按 FlowGate 仓库重新整理。

## 更新

- 增加无试用套餐自动重试能力
- 增加 PayPal 回调自动重试能力
- 增强 PayPal Hosted Checkout 错误识别
- 增强 SESSION JSON 导入重试与超时控制
- 增强 Cloudflare Temp Email 验证码读取兼容性

## 修复

- 修复部分 PayPal Hosted Checkout 页面状态识别问题
- 修复本地 helper 未启动时提示不清晰的问题
- 优化 ChatGPT 注册入口识别
- 修复 PayPal Hosted Checkout 验证码读取兼容性问题
- 减少账号池重复取用
